基本概念

物理层主要任务：
确定与传输媒体接口有关的一些特定==>定义标准
如：

  1. 机械特性(规定物理连接时采用的规格、接口形状、引线数目、引脚个数等)
  2. 电气特性(规定传输二进制时，线路上的电压范围、阻抗匹配、传输速率和距离限制等等)
  3. 功能特性(指明某条线上出现的某一电平表示何种意义，接口部件的信号线的用途)
  4. 规程特性(定义各条物理线路的工作规程和时序的关系)

数据通信基础概念

大概图解

调制解调器相当于家里的猫

术语

数据：传送信息的实体，通常是有意义的符号序列
信号：数据的电气/电磁的表现，是数据在传输过程中的形式

数字信号：代表信息的参数取值是离散的
模拟信号：代表信息的参数取值是连续的

新源： 产生和发送数据的源头
信宿： 接收数据的终点
信道：信号的传输媒介，一般用来表示向某一方向传送信息的介质


三种通信方式

单工通信

只有一个方向的通信，只有一条信道
PS：相当于单方面挨喷

半双工通信

通信双方都能发送或接收信息，但是不能同时发送和接收，两条信道
PS：相当于回合制对喷

双工通信

通信双方可以同时发送和接收信息，要两条信道
PS：无阻碍对喷

两种传输方式


速度快所以用并行传输，适合短距离

物理层的名词

码元



几种高度代表几进制码元
M进制码元带有log2M的bit信息

速率

速率也叫数据率，指数据的传输速率（跟上一章的概念一样），也可以用码元传输速率和信息传输速率表示

码元传输速率

也叫码元速率、波形速率、调制速率、符号速率等，表示单位时间内数字通信系统传输的码元个数(也叫脉冲次数或信号变化次数)，单位是波特(Baud)**，即1波特=1码元/s，但码元传输速率与进制数无关**

信息传输速率

别名信息速率、比特率，表示单位时间内数字通信系统传输的二进制码元个数（即比特数），单位是b/s

  • 关系：若一个码元携带n bit的信息量，则M Baud的码元传输速率对应的信息传输速率为M×n bit/s

带宽

表示在单位时间内从网络中的某一点到另一点所能通过的“最高数据率”，常用来表示网络的通信线路所能传输数据的能力，单位是b/s

理解：我的理解是理论上的理想最高数据率

码间串扰


低频（200HZ）不能通过的原因是传输距离过长会逐渐损耗，高频（4000HZ）不能通过的原因是码元传输速率过快，码元与码元之间靠的太近不容易辨析
需要注意上图的信道带宽的概念

奈氏准则（奈奎斯特定理）重要

奈氏准则：在理想低通（无噪声、带宽受限）条件下，为了避免码间串扰，极限码元传输速率为2W Baud，W为信道带宽，单位是HZ

名词解释：
低通：低于最高频率的，也就是带宽受限的意思

注意：只有这两个公式（奈氏准则和香农定理）中信道带宽的单位是HZ，当题目中提到信道带宽是以HZ为单位时应该联想到要用公式

2W代表极限码元传输速率，log2V代表一个码元携带的bit信息，相乘就得到了极限信息传输速率

应用：

  1. 在任何信道中，码元传输的速率是有上限的。若传输速率超过了上限，就会出现严重的码间串扰问题。
  2. 信道的频带越宽（即能通过的信号高频分量越多），就可以用更高的速率进行码元的有效传输
  3. 奈氏准则给出了码元传输速率的限制，但是没有对信息传输速率给出限制
  4. 由奈氏准则可知，当每个码元携带的比特数更多时，信息传输速率就会更高

例题：

香农定理

香农定理：在带宽受限且有噪声的信道中，为了不产生误差，信息的数据传输速率有上限值。
信噪比=信号的平均功率/噪声的平均功率，通常记为S/N，并且用分贝(dB)作为度量单位，即：信噪比(dB)=10log10(S/N) 他们的数值是等价的
我的理解：所谓信噪比就是信号和噪声的比值，信号强，噪声若则信号清晰，信号弱，噪声强，则信号模糊

推论：

  1. 信道的带宽或信道中的信噪比越大，则信息的极限传输速率越大
  2. 当信道的带宽和信噪比确定之后，信息传输速率的上限就确定了
  3. 只要当前的信息传输速率低于信道的极限传输速率，就一定能找到某种方法来实现无差错传输
  4. 实际信道的传输速率比香农定理的极限传输速率低不少

编码与调制

基带信号与宽带信号


在传输距离较近时，计算机网络采用基带传输方式(近距离衰减小，从而信号内容不易发生变化)
在传输距离较远时，计算机网络采用宽带传输方式(远距离衰减大，信号变化大也能过滤出基带信号)
不同数据的转换：

数字数据编码为数字信号

非归零编码：高1低0，但是没有检错能力，收发双方难以保持同步
归零编码：信号电平要在一个码元内恢复到0
反向不归零编码：信号电平翻转表示0，信号电平不变表示1
曼彻斯特编码：前低后高表示1、前高后低表示0或者相反，特点是每个码元中间出现电平跳变，位中间的跳变可作为时钟信号用于同步，它所占的频带宽度是原始基带宽度的两倍，因此**数据传输速率只有调制速率的1/2**
差分曼彻斯特编码：同1异0，常用于局域网传输，抗干扰性强于曼彻斯特编码
4B/5B编码：

数字数据调制为模拟信号

数字数据调制技术就是：数字信号=>模拟信号==传输==>模拟信号=>数字信号
数字信号转换为模拟信号是调制
模拟信号转换为数字信号是解调

模拟数据编码为数字信号

  1. 最易于理解的例子就是计算机中处理的数字音频，通过将模拟音频采样、量化转换成有限个数字表示的离散序列（即实现数字音频化）
  2. 最常见的编码方式是PCM（脉码调制），也是能够达到最高保真水平的编码
  3. 模拟信号转换为数字信号包括三步，抽样，量化，编码

抽样：对模拟信号进行周期性扫描，从连接变为离散，为保证不失真，使用采样定理：f采样频率>=2f信号最高频率
量化：把抽样得到的电平幅值转换为分级的数字取整，这样就实现了从连续到离散
编码：把上一步的转换为二进制编码

拓展：

而采样频率高也就意味着采样设备要更精确，所以有时候就会发生高频失真

模拟信号调制为模拟信号

物理层传输介质

传输介质以及分类

传输介质也称传输媒体/传输媒介，就是发送设备和接收设备之间的物理通路。

注意：传输媒体并不是物理层，且在物理层下面，传输媒体不知道所传输的信号的含义，而物理层定义了电气特性，因此能识别比特流

导向性传输介质

导向性传输介质–双绞线


网线用的就是双绞线

导向性传输介质–同轴电缆

导向性传输介质–光纤



光纤特点：

  1. 传输损耗小，中继距离长，对远距离传输特别经济
  2. 抗雷电和电磁干扰性能好
  3. 无串音干扰，保密性好，不易被窃听
  4. 体积小，重量轻

非导向性传输介质

物理层设备

中继器

  • 功能：由于信号功率会损耗，因此需要中继器对信号再生和还原，对衰减信号进行放大，增加信号传输的距离，延长网络长度
  • 中继器两端要同一个网段且速率相同，两端要同一个协议，可以是不通同媒体
  • 中继器不会存储转发
